 이전 트윗덱 데스크톱 기능과 다르지 않으면서
 불편했던 점을 보강하는데 중점을 두었습니다.
 
 트윗덱 플레이어에서는 다음 기능이 있습니다
 - 파일 끌어다 놓아 첨부하기
 - 트윗덱으로 동영상이나 움짤 보기
 - 창을 항상 위로 표시하기
 - 외부 링크 팝업 띄우기
 - 폰트 커스텀
 - 자바스크립트 끼워넣기

 트윗덱 플레이어에서 없는 기능도 있습니다
 - 알림 기능 (Notification)

 자바스크립트 실행 기능은 Script 폴더의 script.js
 파일입니다. tweetdeck.twitter.com 에서만 읽습니다
 한글 폰트뿐만 아니라 영문 폰트까지 바꾸려면
 스크립트를 이용해 폰트를 변경해 주세요.
