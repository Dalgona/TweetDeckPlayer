 이전 트윗덱 데스크톱 기능과 다르지 않으면서
 불편했던 점을 보강하는데 중점을 두었습니다.
 
 트윗덱 플레이어에서는 다음 기능이 있습니다
 - 파일 끌어다 놓아 첨부하기
 - 트윗덱으로 동영상이나 움짤 보기
 - 창을 항상 위로 표시하기
 - 외부 링크 팝업 띄우기
 - 폰트를 마음대로 커스텀 (일부 폰트는 지원되지 않습니다)
 - 트윗된 이미지를 다운받을 때 원본 품질(orig)로 다운받기
 - 트위터에서 바로 트윗하기 / 트위터를 팝업으로 열기
 - 스크립트 확장, 스타일 확장 기능 지원 (트윗덱 테마 입히기, 기능 추가..)

 설정파일에서 바꿀 수 있는 것은 다음과 같습니다
 - DefaultAlwaysOnTop : 실행시 윈도우를 항상 위로 설정
 - DisableWriteTweetMenu : 트위터로 트윗하기 메뉴 끄기
 - DisalbeTwitterOpenMenu : 트위터 열기 메뉴 끄기
 - DisableTwimgOrig: 트윗 이미지를 원본 품질로 저장하기 끄기
 - DisableLinkPopup : 링크를 팝업이 아닌 브라우저로 열기
 - DisablePopupOpenMenu : 팝업으로 열기 메뉴들을 끄기
 - fontFamily : 폰트 커스텀 설정

 트윗덱 플레이어에서 없는 기능도 있습니다
 - 알림 기능 (Notification)

 트윗덱에서는 스크립트 확장과 스타일 확장을 지원합니다
 트윗덱 플레이어에 스크립트를 적용시키거나 테마를 변경,
 스타일을 바꾸려면 다음을 참고하세요.

 - 스크립트는 script 폴더의 js 확장자로 된 모든 파일을 읽습니다
 - 스타일은 script 폴더의 css 확장자로 된 모든 파일을 읽습니다
 - userstyles.org 등에서 가져온 커스텀 스타일을 적용하려면
   @-moz-document domain("tweetdeck.twitter.com") 구문을 꼭 지워주세요
 - 한글 폰트뿐만 아니라 영문 폰트까지 바꾸려면 스크립트나 css를 이용해
   폰트를 변경해 주세요.
 - 스크립트 확장과 스타일 확장은 'twitter.com', 'tweetdeck.twitter.com'에만
   적용됩니다
